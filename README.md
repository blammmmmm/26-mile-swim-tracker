# Swim Tracker Overlay

Fixed version:
- Renamed from "Cliffy Swim Tracker" to "Swim Tracker"
- Shorter card so it does not get cut off
- Bottom stats fit inside the overlay
- Compact Apple-style glass UI
- Firebase path: `swimTracker/current`

## OBS settings

Browser Source URL:
`https://blammmmmm.github.io/26-mile-swim-tracker/overlay.html`

Recommended size:
- Width: 760
- Height: 310

If you need a smaller version:
`overlay.html?scale=0.9`

If you want to force opacity:
`overlay.html?opacity=0.65`
